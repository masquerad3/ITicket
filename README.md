# ITicket

A simple web application for submitting and managing IT support tickets. Built with HTML, CSS, JavaScript, PHP, and Microsoft SQL Server.

## Project Goals
- Help employees report IT issues with clear forms (short/checkable + detailed).
- Allow IT staff to view, prioritize, and update tickets.
- Track status and analyze common recurring issues.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: Microsoft SQL Server (sqlsrv driver)
